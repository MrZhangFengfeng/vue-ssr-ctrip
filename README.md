# vue-ssr-ctrip

> 基于nuxt全栈实现携程旅行网核心功能

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
## 基本技能点：
- Vue全家桶
- Nuxt
- 骨架屏
- Koa2
- MongoDB
- Mongoose
- redis
- Element-UI
- POI
- SASS

## 基本功能点
- 城市选择
- 地图服务
- 定位服务
- 注册/登录  邮箱验证
- 服务端存储
- 推荐搜索服务
- 火车票/机票/酒店搜索服务
- 火车票/机票/酒店下单流程
- 埋点/日志记录
